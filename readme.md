## Build

Clone the repo
```shell
git clone https://github.com/yosbelms/ramdascript-todomvc.git
```

Intall dependencies
```shell
npm install
```

Install Webpack
```shell
npm install -g webpack
```

Run Webpack
```shell
webpack
```

At this point you must serve `index.html`

RamdaScript source-code resides in `src/`.
